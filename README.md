## Requirements
1. openssl
2. curl

## How to Install in Termux

`$ pkg up -y`

`$ pkg install openssl-tool`

`$ pkg install curl`

`$ pkg install git`

`$ git clone https://github.com/D4rk-B0y/InsFollow`

`$ cd InsFollow`

`$ chmod +x InsFollow.sh`

`$ termux-wake-lock`

`$ bash InsFollow.sh`
